Technologies utilisées

Ce site est une application web réalisée à l'aide du framework Flask de Python et le toolkit opensource SQLAlchemy
Les données sont stockées dans une base de données relationnelle, MariaDB
La partie frontend a été réalisée avec Bootstrap, html5, css, javascript, Jquery Le déploiement en production sur les serveurs de l'université s'est fait à l'aide des clients Gunicorn, Apache et Supervisor.
La mise en oeuvre du serveur a été réalisée par la Direction Opérationnelle des Systèmes d'Information de l'université.
Pour la création des notices "lieux", nous utilisons les services web de GeoNames
